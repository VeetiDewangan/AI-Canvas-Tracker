# AI-Powered Virtual Paint 🎨
A Computer Vision application developed using OpenCV and Python that tracks color-specific objects to draw on a digital canvas.

## 🛠 Features
- **Real-time Tracking:** Uses HSV color space for robust detection.
- **Multi-Stroke Support:** Handles continuous drawing and "Pen Up" logic.
- **Mirror View:** Flipped frame for intuitive user interaction.

## 🚀 Setup Instructions
1. Activate your environment: `venv\Scripts\activate` (Windows)
2. Install requirements: `pip install opencv-python numpy`
3. Run the app: `python main.py`

## ⌨️ Controls
- **'c'**: Clear Canvas
- **'q'**: Quit Application
